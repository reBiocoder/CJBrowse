标题¶

  • 一级标题在文本的第二行使用 # 来标记，就像手打下划线一样。

  • 二、三、四、五、六级标题分别使用 * + = : -，通常记住 # * + 就够了。

  • 标题递进必须按顺序，不能一级标题后紧跟三级标题，否则会渲染失败。

  • 原则上这个手打的“下划线”的长度至少跟标题文字一样长，可以超长。

二级标题¶

“下划线”特别长，没关系。
